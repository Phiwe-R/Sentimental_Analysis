## Overview
SentimentScope is a machine learning project that uses a **transformer-based model** to classify IMDB movie reviews as **positive or negative**. The project demonstrates how sentiment analysis can improve content recommendation systems by better understanding user feedback.

## What This Project Demonstrates
- End-to-end **NLP pipeline**: data preprocessing, training, validation, and testing  
- Fine-tuning a **transformer model** for sentiment classification  
- Efficient data handling using **PyTorch DataLoaders**  
- Model evaluation on unseen data for real-world performance  

## Tech Stack
- Python  
- PyTorch  
- Transformers  
- IMDB Dataset  

## Impact
Accurate sentiment classification supports more personalized and relevant movie recommendations, improving user engagement.
